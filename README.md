# Vélour Home - Premium Scandinavian E-Commerce

Vélour Home adalah website e-commerce furniture minimalis-premium bergaya Nordik/Skandinavia yang dibuat menggunakan HTML5, Vanilla CSS, dan JavaScript modern (ES6). Desain situs web ini mengusung estetika premium, bersih, mendukung Dark/Light Mode, dan fungsionalitas keranjang belanja interaktif yang cepat serta ringan.

## Fitur Utama

- **Desain Skandinavia Premium**: Tipografi elegan menggunakan Google Fonts (*Playfair Display* & *Inter*) serta skema warna hangat.
- **Keranjang Belanja Dinamis**: Pengguna dapat menambahkan produk, mengubah jumlah barang, dan menghapus produk langsung dari drawer keranjang belanja.
- **Simulasi Checkout**: Menyediakan form detail alamat pengiriman, opsi pembayaran, dan halaman sukses transaksi beserta nomor invoice.
- **Dukungan Dark Mode**: Fitur pergantian tema (dark/light) yang tersimpan di LocalStorage pengguna.
- **Desain Responsif**: Sangat optimal diakses dari perangkat mobile, tablet, maupun desktop.

## Struktur Proyek

```bash
├── index.html        # Struktur HTML utama dan modal-modal
├── style.css         # Desain sistem, warna HSL, glassmorphism, & animasi
├── products.js       # Database mockup produk furniture
└── app.js            # Logika keranjang belanja, pencarian, filter, & modal
```

## Cara Menjalankan

1. Clone repositori ini ke komputer Anda:
   ```bash
   git clone <URL_REPOSITORI_ANDA>
   ```
2. Buka direktori proyek:
   ```bash
   cd ecommerce-web
   ```
3. Klik dua kali pada file `index.html` untuk menjalankannya langsung di browser Anda.
